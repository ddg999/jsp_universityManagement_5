## ❓ 배경 및 목적 
**<u>프로그래밍 수업을 통해 학습한 다양한 내용을 복습</u>**   
**<u>협업을 이해하고 프로젝트가 완성되는 전 과정을 체험</u>**   

## 주요기능
- Spring Boot, MyBaits, JSP 그리고 MySQL을 이용하여 개발된 웹 기반의 학사 관리 시스템
- 학사 업무를 체계적으로 관리하고 학사 정보를 효율적으로 처리
- 사용자 친화적인 UI 및 다양한 기능을 제공하여 학사 관리 업무를 간편하게 수행

<br>
